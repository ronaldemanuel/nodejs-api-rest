# Node JS API REST

Uma REST API feita utilizando JavaScript e [Node JS](https://nodejs.org/en/)

## Tema base: API para uma PetShop

A ideia é aprender sobre Node JS com [Express](https://www.npmjs.com/package/express) criando do zero
um Web Service para ser consumido por uma aplicação de um PetShop

### Tecnologias utilizadas

- Node JS
- Express
- MySQL
